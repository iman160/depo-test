Aujourd'hui j'ai appris les commandes unix

J'ai egalement commencer a apprendre git un logiciel de programmation

  • mkdir 'nom du fichier' = creer un nouveau dossier'
  • ls lister le contenu de ou on est
  • cd 'nom du fichier' = revenir dans le dossier precedent
  • rmdiv 'nom du fichier' = supprimez un dossier
  • touch 'nom du fichier' = creer un nouveau fichier vide
  • cat 'nom du fichier' = afficher le contenu d'un fichier
